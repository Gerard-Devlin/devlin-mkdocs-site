## 1.3.1

### 第一个C程序
=== "C"
    ```c
    int main()
    {
        print("Hello World!\n");
        
        return 0;
    }
    
    //printf:打印字符串
    //\n: 	换行
    ```
=== "C++"
    ```c++
        int main()
        {
            cout<<"Hello World!"<<endl;
            
            return 0;
        }
        
        //cout 打印字符串
        //endl: 换行
    ```
